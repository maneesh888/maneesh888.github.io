Privacy-First AI Keyboard

Open Keyboard

An iOS keyboard prototype for AI-assisted writing, with grammar correction and broader writing actions available directly inside the custom keyboard extension.

Open Keyboard is the client experience: a custom iOS keyboard where you can type normally and ask for AI help when you need grammar fixes, improvements, rephrasing, or summaries.

The privacy direction is that normal typing stays on the device. Text is sent out only when the user chooses an AI action, and it goes to the gateway they configure.

Current status: active prototype. Recent work completed the first usable AI writing workflow inside the keyboard extension, separated grammar correction from broader writing tools, and improved reliability around loading and text replacement behavior.

Latest Product Progress

Writing

AI Writing Workflow

Implemented AI writing workflows inside the keyboard extension, replacing the earlier placeholder experience with generated suggestions, selected-operation state, and explicit accept behavior.

  • Improve runs through the configured AI path.
  • Suggestions include retry, copy, and accept/replace actions.
  • Rephrase and Summarize remain part of the same writing-action surface.
Grammar

Grammar Correction Workflow

Grammar and typo correction now has its own product flow, separate from broader AI writing tools such as Improve, Rephrase, and Summarize.

  • Starting grammar review requests corrections for the current text.
  • Results can show corrections, a no-issues state, or a visible error state.
  • Grammar review remains available unless configuration, network access, or unrecovered AI/parsing errors block the request.
Quality

Completed Reliability Fixes

Fixed the manual grammar loading and replacement bugs that were blocking reliable correction review.

  • Manual grammar requests are tracked independently from background analysis.
  • Cancelled automatic analysis no longer leaves manual review stuck loading.
  • Single-character article corrections now respect word boundaries and avoid stale duplicate replacements.
Coverage

Deterministic Preview And Tests

The preview lab and regression tests were updated for the new Improve UI and the corrected grammar workflow behavior.

  • Preview support covers the new AI writing panel states.
  • Regression tests cover grammar loading and correction replacement behavior.
  • Focused real-extension smoke coverage verifies the configured keyboard AI controls.
